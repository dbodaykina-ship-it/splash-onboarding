# Finam X · Онбординг (splash) — прототип

Интерактивный прототип splash-онбординга (4 шага: Консьерж → Инвест-аналитик →
Риск-агент → финал + переход в экран чата). Один self-contained файл.

## Доступ
`index.html` защищён паролем (контент зашифрован AES-256-GCM, в открытом виде в файле его нет).
Пароль сообщается отдельно.

## Как смотреть
- Локально: открыть `index.html` в браузере, ввести пароль.
- Онлайн: включить GitHub Pages (Settings → Pages → branch `main` / root), ссылка появится там же.

> Незашифрованный исходник и рабочие ассеты в репозиторий не коммитятся (см. `.gitignore`).
